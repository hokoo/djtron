# Users

В этой папке хранятся учетные записи для удаленных клиентов.

- Один файл = один пользователь.
- Имя файла = логин (например `operator.txt` или `operator`).
- Первая непустая строка в файле = пароль.
- Строки, начинающиеся с `#`, считаются комментариями.

Пример файла `operator.txt`:

```
myStrongPassword123
```

Локальный клиент на том же ПК, где запущен сервер, может работать без логина.
